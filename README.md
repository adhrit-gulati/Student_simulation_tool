# Student_simulation_tool
This is a simulation tool i am designing for educational purposes. It will contain all manner of rigidbody, electric field, current simulations to illustrate various physics concepts.
The end goal for this project is to be able to simulate the usual physics questions asked in books.
Built using python and arcade module
